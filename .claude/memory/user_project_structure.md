---
name: Project folder structure
description: All Claude Code projects should be created in ~/projects/ — each project gets its own subdirectory
type: feedback
---

All projects live in `~/projects/`. When starting a new project, create a subdirectory there (e.g., `~/projects/brainos/`, `~/projects/photo-dropper/`). Never scaffold projects directly in the home directory.

**Why:** User works on many different projects with Claude Code and wants a clean, organized home directory.

**How to apply:** When creating any new project, always use `~/projects/<project-name>/` as the root.
